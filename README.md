# Aaron & Tanya Wedding Website

A static, mobile-friendly wedding invitation designed for GitHub Pages.

## Replace the photos

Place your real JPG images inside the `images` folder using these exact filenames:

- `hero.jpg` — opening full-screen image; landscape works best
- `venue.jpg` — Oak Knoll Village Clubhouse photo
- `photo-01.jpg` through `photo-08.jpg` — collage images

Recommended export size: 1800–2400 pixels on the longest side, JPG quality around 75–85%.

## Publish with GitHub Pages

1. Create a new public GitHub repository, such as `aaron-tanya-wedding`.
2. Upload all files and folders from this package to the repository root.
3. Open repository **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.
6. GitHub will provide a URL similar to `https://yourusername.github.io/aaron-tanya-wedding/`.

## Update event information

Open `index.html` in a text editor and search for the wording you want to change.

The RSVP link appears three times. Search for:
`docs.google.com/forms`

The Google Maps link appears once. Search for:
`google.com/maps`

## Custom domain

A domain is optional. GitHub Pages is perfectly suitable for this site. A custom domain can be connected later without redesigning anything.
